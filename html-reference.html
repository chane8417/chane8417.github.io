HTML Reference Page

What Constitutes a Valid HTML Page

A valid HTML Page first requires a <!DOCTYPE html> at the top, and below that a <html> with an end tag at the bottom. Without these two tags, the computer won't know that this is html code.

A <head> tag is required too to store metadata. Also, in order to actually make the website show something, a <body> is needed. The elements that show something physically on the website are usually put in the body. An end tag for the body element is also needed.


The HTML Element

Description:The HTML element is the root element of an HTML document. Without it, the code will not work.

Tag:<html>

Placement:The HTML element is usually placed at the top of the page, right below the <!DOCTYPE html> element. The end tag is placed at the bottom of the page.

Example:The HTML element is used to show that the coding language is HTML.

Rendering:This webpage's content won't be here without the HTML element.

The Body Element

Description:The Body element contains all the webpage's content, like the words and images and links.

Tag:<body>

Placement:The tag is placed below the <html> element.

Placement:The Heading element is placed inside the body element.
