Examples of the 3 different selectors

Element name

p {
color:red;
font-family:verdana;
}

h1 {
color:green;
font-family:courier
}

By using that code, all the paragraphs would become the style that is marked with a p. All the heading 1s would have the style that is marked with a h1. It would look like this:

Color is green and font family is courier

Color is red and the font is verdana

Id

#id1 {
color:violet;
background-color:red;
}

#id2 {
color:orange;
background-color:green;
}

In order to use ids, you also need to add something in front of each paragraph. Like this:

<h1 id=id1>Heading 1</h1>

Depending on which style you want to use, whether the styles under id1 or id2, all you have to do is add a id=id1 to the element. If you assigned id1 to the heading and id2 to the paragraph, it would look like this:

Color is violet and background color is red

Color is orange and background color is green

Class

.pinkc {
color:pink;
text-align:center;
}

.bluer {
color:blue;
text-align:right;
}

You can name the word in front of the { whatever you want, but it needs to begin with a ..Like the id, you need to add class=pinkc after the <h1

If we assign the pinkc to the heading and bluer to the paragraph, it will look like this:

Color is pink and the text is center aligned.

Color is blue and text is right aligned.
